# Data Documentation

## Overview

This folder contains supplementary transparency materials for the manuscript:

**Employee experience in the organizational socialization process: A generational phenomenological analysis**

The study used an interpretative phenomenological approach to examine how newcomers construct meaningful organizational integration during onboarding. The analytical focus was placed on employee experience, anticipatory expectations, perceived social support, role clarity, belonging, and generational differences.

## Data status

The original qualitative narratives are not publicly available. The open-ended responses may include contextual information about organizations, work roles, supervisors, teams, locations, or career trajectories. Even after removal of direct identifiers, such narratives may carry residual re-identification risk.

For this reason, the repository does not include the raw empirical corpus.

## Files included

### `aggregate_coding_matrix.csv`

This file summarizes the main analytical categories and their generational intensity levels. It includes the category label, category description, intensity by generational cohort, interpretative dimension, relational or cognitive mechanism, and theoretical linkage.

### `codebook.csv`

This file describes the variables used in the supplementary transparency package. It provides variable names, labels, variable types, allowed values, and anonymization or synthesis notes.

### `synthetic_participant_example.csv`

This file is a synthetic illustrative example. It does not contain real participant-level data. It was created to show how the analytical structure can be represented at participant level without disclosing the original qualitative responses.

### `anonymization_log.md`

This file explains the anonymization rationale, synthesis process, and ethical limitations associated with public sharing of qualitative onboarding narratives.

## Recommended use

These materials may be used to understand the coding logic, analytical structure, and interpretative architecture of the study. They should not be used to reproduce the original qualitative analysis as if they were the raw empirical corpus.

## Recommended data availability statement

The original qualitative narratives are not publicly available because the open-ended onboarding accounts contain confidential and potentially identifying contextual information. To enhance analytical transparency, the authors provide an aggregate coding matrix, a codebook, an anonymization and synthesis log, and a synthetic participant-level example file reconstructed from the reported analytical structure of the study.

## Ethical note

Qualitative onboarding narratives can contain indirect identifiers even when direct identifiers are removed. For this reason, the public sharing of raw narratives was avoided to protect participant confidentiality and organizational anonymity.
