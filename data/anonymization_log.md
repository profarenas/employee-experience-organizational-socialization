# Anonymization and Synthesis Log

## Purpose

This document explains the anonymization and synthesis rationale adopted for the public transparency materials associated with the manuscript:

**Employee experience in the organizational socialization process: A generational phenomenological analysis**

## Original empirical material

The original study was based on qualitative onboarding narratives provided by participants. These narratives were analysed through an interpretative phenomenological approach.

The original open-ended responses are not publicly shared because they may contain indirect identifiers, including references to organizations, supervisors, teams, work roles, locations, career trajectories, or highly specific onboarding situations.

## Public materials

The repository includes public transparency materials rather than the original qualitative corpus. These materials include:

* an aggregate coding matrix;
* a codebook;
* a synthetic participant-level example file;
* this anonymization and synthesis log.

## Anonymization rationale

Direct identifiers such as names, emails, telephone numbers, organizations, locations, supervisors, and specific institutional references are not included in the public files.

Quasi-identifiers such as exact age, exact dates, precise job titles, organizational names, and highly specific workplace circumstances are also excluded.

The repository uses broad analytical categories and generational cohorts only.

## Synthetic example file

The file `synthetic_participant_example.csv` is synthetic. It does not contain real participant-level data and should not be interpreted as an anonymized version of the original dataset.

The purpose of the synthetic file is to illustrate how the analytical variables can be represented structurally without disclosing the original qualitative narratives.

## Aggregate coding matrix

The file `aggregate_coding_matrix.csv` provides a transparent representation of the thematic and interpretative architecture of the study. It summarizes analytical categories, generational intensity levels, interpretative dimensions, mechanisms, and theoretical linkages.

## Ethical limitation

Open-ended qualitative narratives can remain identifiable even after removing direct identifiers. For this reason, public dissemination of the original empirical corpus was avoided.

## Recommended citation statement

The original qualitative narratives are not publicly available due to confidentiality and re-identification risks. Publicly available repository materials include an aggregate coding matrix, a codebook, an anonymization and synthesis log, and a synthetic participant-level example file designed to document the analytical structure of the study.
