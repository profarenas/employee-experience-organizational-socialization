# Employee Experience in the Organizational Socialization Process

This repository contains supplementary transparency materials for the manuscript:

**Employee experience in the organizational socialization process: A generational phenomenological analysis**

The study examines how meaningful organizational integration is phenomenologically constructed during onboarding, with particular attention to employee experience, anticipatory expectations, perceived social support, and generational differences.

## Repository purpose

This repository is intended to support analytical transparency. It does not contain the original raw qualitative corpus. The original study was based on open-ended onboarding narratives provided by participants. These narratives are not publicly shared because they may contain contextual information that could increase the risk of participant, organizational, or occupational re-identification.

Instead, this repository provides:

* an aggregate coding matrix;
* a variable codebook;
* an anonymization and synthesis log;
* a synthetic participant-level example file illustrating the analytical structure used in the study.

These materials are designed to document the analytical architecture of the research, not to replace the original empirical corpus.

## Important data notice

The file `synthetic_participant_example.csv` is synthetic. It should not be interpreted as the original raw participant-level dataset. It was created only to illustrate the structure of the analytical variables and the logic of the coding framework.

The file `aggregate_coding_matrix.csv` provides the main transparent summary of the coding structure reported in the manuscript.

## Data availability statement

The original qualitative narratives are not publicly available because the open-ended onboarding accounts contain confidential and potentially identifying contextual information. To enhance analytical transparency, this repository provides an aggregate coding matrix, a codebook, an anonymization and synthesis log, and a synthetic participant-level example file reconstructed from the reported analytical structure of the study.
